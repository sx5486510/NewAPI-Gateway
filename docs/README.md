## 文档维护约定

1. 所有 `docs/*.md` 顶部保留“返回文档入口”链接。
2. 每篇文档底部至少提供 2 个“相关文档”反向链接，避免孤岛文档。
3. 文档新增后必须同时更新本页“完整文档清单”与“联动维护矩阵”。
4. 链接统一使用相对路径，保证仓库离线可读。
